# API Contract Diff
- 新增 `POST /internal/provision`：request `{"qq_uin":"123","display_name":"Alice"}`；response 200 `{"status":"pending|active","secret":"...","user_id":"..."}`；status 400/401/503
- 新增 `POST /internal/confirm`：request `{"qq_uin":"123"}`；response 200 `{"status":"active","user_id":"..."}`；status 400/401/404
- 新增 `GET /internal/user_status?qq_uin=...`：response 200 `{"status":"pending|active|disabled|null","display_name":"...","last_login_at":"..."}`；status 400/401
- 新增 `GET /internal/rank?qq_uin=...`：response 200 `{"rank":"unranked","mmr":0,"wins":0,"losses":0}`；status 400/401
- 新增 `POST /auth/login`：request `{"qq_uin":"123","secret":"..."}`；response 200 `Set-Cookie: session=...` + `{"status":"ok","user_id":"..."}`；status 400/401
- 内部契约变更：Y（新增 ProvisionRequest/ConfirmRequest/LoginRequest）
