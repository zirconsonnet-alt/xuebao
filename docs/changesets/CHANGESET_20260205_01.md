# ChangeLog
- 将交互式流程 `run_flow` 从 `src/plugins/nonebot_plugin_manager/utils.py` 收敛到 `src/interfaces/nonebot/flow_runtime.py`，application 服务不再依赖 deprecated 兼容层。
- 移除 `src/application/group.py` 的 `Group.bot`（不再直接 `nonebot.get_bot()`），改为 `Group.self_id` 与 `Group.set_special_title()` 通过 `GroupGateway` 调用。
- 统一 application/services 中对 `wait_for`/`wait_for_event`/`get_name*` 等工具函数的引用到 `src/interfaces/nonebot/utils.py`。
- 将 AI 助手所需的文本清理与知乎抓取能力从 deprecated 兼容层拆出：`src/core/text_processing.py`、`src/infrastructure/external/zhihu_client.py`。
- `CompositionService` 不再动态注册 `on_message` matcher，改为 waiter 循环等待并通过 `Group.set_msg()` 设置精华，减少服务层对 NoneBot 全局对象的直接依赖。
